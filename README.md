# PatrolEDR

**PatrolEDR** is a lightweight research-focused Endpoint Detection and Response (EDR) tool for Windows.  
It monitors process creation in real-time and can detect suspicious processes based on parent-child relationships.

## Features

- Real-time monitoring of process creation
- Detects suspicious processes launched by untrusted parents
- Logs alerts to console (future: file or server logging)
- Lightweight and easy to extend for research purposes
